      제시된 🪱 도형의 변환 규칙은 파악하여 물음표에 해당하는 도형을 유추하는 유형의 문제
      ⏳ 35 초 이내에 풀어야 함.
      🔰
        Tip by type
          * 박스형 문제
            제시된 도형들 간의 규칙이 열과 열 사이에 적용되는지, 행과 행 사이에 적용되는지 정확히 파악하는 연습을 한다.
              - (행 | 열) 의 모든 칸에 적용되는 것이 아니라도 규칙으로 볼 수 있다!
              - (1,1), (1,2) 와 같은 결과 칸 자체에는 아무런 의미가 없을 수도 있다. 
              (1, 1) ⋆ (1, 2) = (1,3)
            해당 칸 안에있는 도형 자체를 대수처럼, 또는 집합처럼 취급해서, 수식을 만들어보자!
              🛍️ e.g.
                - A + B = C
                - A ∩ + B = C
            
      🛍️ e.g. 제시된 도형의 시계 또는 반시계 방향으로 회전, 색반전, 위치 이동 후 색반전, 회전 후 색반전, 내부 도형 시계 방향으로 회전+외부 도형 반시계 방향으로 회전, 상하 대칭 또는 좌우 대칭, 칸 이동, 1열과 2열의 결합, 공통 음영 색칠
