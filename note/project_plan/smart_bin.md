# 스마트 쓰레기통 관리 시스템

스마트 쓰레기통 관리 시스템은 쓰레기통이 가득 차면 알림을 보내고, 비접촉 방식으로 뚜껑을 자동으로 열고 닫을 수 있어 위생적입니다. Zigbee와 Matter를 통해 스마트 홈 네트워크와 통합하여 편리한 관리가 가능합니다.

---

## 프로젝트 개요
- **목적**: 사용자의 위생과 편의성을 높이기 위해 쓰레기통이 일정 수준 이상 찼을 때 알림을 보내고, 비접촉 방식으로 뚜껑을 열고 닫는 자동화된 시스템입니다.
- **프로토콜 선택**:
  - **Zigbee**: 쓰레기통 상태 모니터링 및 뚜껑 자동 개폐 기능에서 저전력으로 데이터를 주고받아, 에너지를 절약하면서도 효율적으로 반응할 수 있습니다.
  - **Matter**: 스마트 홈 기기와 연동하여 스마트폰 알림 전송 및 다른 스마트 기기와 통합을 지원합니다.

---

## 주요 기능
1. **쓰레기통 상태 모니터링**
   - **초음파 센서**를 통해 쓰레기통 내부의 쓰레기 높이를 측정하여 가득 찼는지 여부를 확인합니다.
   - Zigbee 네트워크를 통해 쓰레기통 상태를 주기적으로 모니터링하며, 가득 찼을 경우 알림 신호를 전달합니다.

2. **자동 뚜껑 개폐**
   - **서보 모터**로 비접촉 방식의 뚜껑 개폐 기능을 제공합니다.
   - Zigbee 네트워크를 통해 사용자가 쓰레기통에 접근할 때 센서로 이를 감지하여 즉시 뚜껑이 열리고 일정 시간이 지나면 자동으로 닫힙니다.

3. **알림 기능**
   - 쓰레기통이 가득 차면 Zigbee로 상태를 전송하고, 라즈베리 파이에서 Matter 프로토콜을 통해 스마트폰 알림을 보냅니다.
   - Matter 프로토콜을 통해 알림을 사용자 스마트폰이나 스마트 홈 기기로 전송하여 상태를 실시간으로 확인할 수 있습니다.

4. **스마트 홈 연동**
   - Matter를 통해 Apple HomeKit, Google Home, Amazon Alexa 등 주요 스마트 홈 생태계와 연동하여 사용자가 쓰레기통 상태를 확인하고, 음성 명령을 통해 상태 확인이 가능합니다.
   - 쓰레기통이 가득 찬 경우, 스마트 조명과 연동하여 색상을 변경하거나 다른 기기로 시각적 알림을 제공할 수 있습니다.

---

## 아키텍처
- **아두이노**:
  - **초음파 센서**로 쓰레기통 내부 높이를 측정하여 가득 찼는지 여부 확인.
  - 가득 찬 상태를 **Zigbee 모듈**을 통해 전송하여 라즈베리 파이로 상태를 전달.
- **라즈베리 파이**:
  - **MySQL 서버**에 쓰레기통 상태와 비운 시기를 기록하고 관리.
  - Zigbee 신호를 받아 상태를 기록한 후, Matter 프로토콜을 통해 스마트폰에 알림 전송.
- **STM32**:
  - **서보 모터** 제어로 뚜껑 자동 개폐 기능 수행.
  - 일정 시간 동안 사용자가 접근할 경우 Zigbee 네트워크를 통해 뚜껑을 열고, 일정 시간이 지나면 자동으로 닫음.

---

## 시스템 흐름
1. **쓰레기 상태 감지**:
   - 아두이노의 **초음파 센서**가 쓰레기통 내부의 쓰레기 높이를 측정하여, 쓰레기가 가득 차면 Zigbee 네트워크로 상태 전송.

2. **알림 전송 및 기록**:
   - 라즈베리 파이는 Zigbee로 전달된 가득 찬 상태를 MySQL 서버에 기록하고, Matter를 통해 스마트폰으로 알림을 전송.

3. **뚜껑 자동 개폐**:
   - 사용자 접근 시, STM32가 Zigbee를 통해 서보 모터를 제어하여 뚜껑을 자동으로 열고, 일정 시간이 지나면 닫음.

4. **스마트 홈 연동**:
   - Matter 네트워크를 통해 라즈베리 파이가 스마트 홈 기기와 연동하여 사용자에게 알림을 전송하고, 음성 비서나 스마트 조명으로 쓰레기통 상태를 시각적으로 확인할 수 있도록 통합.

---

## 추가 고려사항
- **에너지 절약**: Zigbee의 저전력 특성을 활용하여 배터리 사용 시간을 극대화하고, 사용하지 않을 때는 센서와 모터의 전력을 최소화.
- **위생적 사용**: 비접촉 방식으로 쓰레기통을 열고 닫아 위생적이며, 실내 공기 오염을 방지하기 위해 탈취제 모듈 추가 고려 가능.
- **배터리 관리**: 배터리로 구동 시 저전력 모드를 통해 수명을 최적화하고, 주기적인 상태 점검 및 충전 알림 기능 추가 가능.

---

## 예상 결과물
이 시스템을 통해, 사용자는 실내에서 쓰레기통을 비접촉 방식으로 위생적으로 사용할 수 있으며, 가득 찼을 때 자동으로 알림을 받아 편리하게 관리할 수 있습니다. Zigbee와 Matter를 통한 스마트 홈 연동을 통해 더욱 통합된 스마트 홈 경험을 제공합니다.
